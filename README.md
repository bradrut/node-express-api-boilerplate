# node-express-api 

Boilerplate code to use when building a new REST API using Node.js and Express, including relevant tools and middleware.

Includes:
- Express.js configuration
- Sample end-to-end endpoint:
  - Boilerplate `controller` endpoint with request/response types
  - Boilerplate `service` layer
  - Boilerplate `model` layer
  - API route definition registered with Express
- Custom error handling classes and middleware
  - With `express-async-errors` configuration to handle async errors
- tsx (for running the app with live reload)
- dotenv
- `.gitignore`
